美伊停火談判陷入「霍爾木茲方程式」僵局——Trump拒接受停火，歐亞市場強彈

發生什麼事

2月底美以聯軍對伊朗發動打擊以來，美伊衝突持續5週。4月1日，Trump公開表示伊朗已透過中間人要求停火，但美方前提條件是霍爾木茲海峽必須完全重開；伊朗外長Araghchi接受Al Jazeera採訪時確認雙方有訊息往來（透過特使Witkoff），但明確否認進入正式談判，稱「沒有談判、沒有信任」。Trump另表示美軍最快「二至三週內」可以撤離。受此訊號提振，Brent油價由高點回落至每桶約$100，歐洲股市強勢反彈，韓股領漲亞洲，10年期美債殖利率幾乎持平。俄羅斯同日宣布禁止所有汽油出口，令全球供應面額外承壓。

📐 解讀：市場對「川普放話」已呈現明顯的敏感性遞減——3月Brent暴漲60%（1988年來最大月漲幅），今天油價卻僅因言論小幅回落，說明市場逐步price in「川普動口不動手」模式。真正分歧在於霍爾木茲海峽能否在無地面部隊情況下恢復通航；目前伊朗外長的回應顯示雙方距離實質協議仍有相當距離。
🔗 霍爾木茲停航 → 全球20%石油/LNG供應中斷 → 能源通膨輸入型上行 → 各國央行降息空間壓縮 → 高負債企業/REITs/新興市場再融資壓力升高 → 基礎建設投資風險溢價上行
🏗️ 結構性 — 結構：美以聯手對抗伊朗核武計劃是地緣結構性重組；油價波動本身帶有週期特徵
⚡ 矛盾訊號：Trump說「伊朗求和」vs 伊朗外長說「沒有談判沒有信任」；市場上漲 vs IEA說4月供應缺口將惡化——兩者方向矛盾，其中一方在誤判，兩週內將收斂。
🔀 如果霍爾木茲在4月中前基本恢復通航，則Brent可回落至$85-90，全球通脹路徑回歸正常，聯準會6月重啟降息討論；如果戰事延續至5月以後、伊朗升級打擊海灣國家基礎設施，則Brent可能重破$120，各國央行被迫在衰退壓力中維持鷹派立場，Q2出現stagflation情境。

美國租金年跌1.7%（2017年來最大降幅）+ 抵押貸款再融資需求月跌40%——戰爭恐慌重壓房市

發生什麼事

Apartment List數據顯示2026年3月美國公寓租金年比下跌1.7%，是該機構自2017年開始追蹤以來的最大年跌幅，甚至超過2020年COVID初期的跌幅，原因是伊朗戰爭引發的經濟恐慌導致租房需求萎縮，疊加裁員潮。同期CNBC數據顯示，過去一個月美國每週房貸再融資需求暴跌40%以上，主因是能源通膨推高利率預期導致房貸利率上升，加上消費者信心受損。美國住宅市場呈現「量縮、房租跌、貸款需求驟降」三重壓力同步出現。

📐 解讀：這是一個典型的需求面衝擊——不是供給過剩引起的租金下跌（2020年是供需均衡問題），而是純粹的需求萎縮（消費者因戰爭恐慌縮減開支）。這與住宅REITs估值和開發商盈利前景形成直接壓力，但若停火落地，需求可能快速恢復。
🔗 伊朗戰爭恐慌 → 消費信心下滑 → 住房搜索需求減少 → 租金年跌1.7% → 公寓REITs NAV下修 → 房貸利率上升（能源通膨推高殖利率）→ 再融資需求跌40% → 建築相關貸款違約率上升 → 銀行放款謹慎
🏗️ 結構性 — 週期：目前下跌是恐慌性需求萎縮（可逆），非結構性住宅過剩；但若戰事延長超過6個月，部分城市租金跌幅將轉為結構性
⚡ 矛盾訊號：租金下跌1.7%（需求萎縮）vs 建築成本上升（材料+能源通膨）——開發商兩端受壓，利潤率嚴重壓縮，但銀行估值尚未完全反映此風險。
🔀 如果停火在4月底完成，消費信心快速恢復，租金下跌可能在Q3逆轉，REITs估值重評；如果戰事延至Q3，則主要城市租金累計跌幅可能達到-3% to -5%，部分辦公室/商業REIT面臨再融資壓力。

🏗️ 結構性 — 結構：供應鏈重組是對地緣政治永久性調整的結構性響應；但關稅政策本身具有週期性（受美國選舉影響）
⚡ 矛盾訊號：57%台企正在重組供應鏈（去風險趨勢）vs 台灣被稱為「利基超級大國」（對全球不可缺）——兩者同時為真，說明台灣的戰略地位已讓其有談判籌碼，但也增加了地緣脆弱性。
🔀 如果美台FTA在2026年取得實質進展（機率偏低），則台灣供應鏈重組壓力顯著緩解，高科技出口可維持雙位數增長；如果川普對等關稅全面實施，則台灣出口增速可能從+15%降至+5-8%，台幣面臨貶值壓力。

短期（1個月內）：短期（1個月內）：台灣PMI 55.4%顯示製造業仍健康，但趨緩信號值得關注；外資回補232億強化短期支撐，加權指數在33,000-34,000點區間震盪。

長期（1年以上）：長期（1年以上）：台灣「利基超級大國」地位使其成為全球最不可或缺的供應鏈節點，TSMC、CoWoS、先進封裝的稀缺性護城河持續加深，長期估值中樞上移。
