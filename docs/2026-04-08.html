美伊兩週停火達成 — 全球資產大規模重定價，WTI暴跌逾15%至95美元

發生什麼事

美國總統川普於2026年4月7日晚間宣布與伊朗達成兩週停火，並允諾重開荷莫茲海峽，消息推動WTI原油期貨單日跌逾15%至每桶95美元，創六週戰爭中最大單日跌幅。全球股市同步反彈：歐股開盤漲4%、旅遊股領漲7%，那斯達克期貨跳漲逾3%，黃金小幅上揚（地緣風險溢價並未完全消退）。停火由巴基斯坦總理夏立夫斡旋，川普在自設截止期限數小時前拍板，協議框架包含10點計畫，伊朗方面將核材料「挖出移走」。然而停火即刻面臨挑戰：以色列未納入協議持續轟炸黎巴嫩真主黨，伊朗八小時後以此揚言退出協議，荷莫茲海峽通行量仍低，伊朗革命衛隊警告「隨時可開火」。

📐 解讀：這波反彈是情緒面主導的去風險逆轉，而非基本面的永久改善。六週戰爭已造成全球能源供應結構性衝擊——Brent現貨一度達141美元（2008年以來最高）、油價漲逾60%——兩週暫停無法修復供應鏈重新布局的決策。更深層的訊號是：市場同時買股票又買黃金（通常反向），顯示投資人對停火可持續性存疑，在享受鬆一口氣的同時繼續保留避險。

發生什麼事

美伊停火消息令美債殖利率大幅下行，CME FedWatch顯示市場對2026年聯準會至少一次降息的概率跳升至43%，為戰爭爆發六週以來最高。原因是WTI油價急跌至95美元，能源通膨壓力有望緩解。與此同時，紐西蘭央行維持基準利率於2.25%不變，明確表示「忽略燃油價格飆升帶來的初期通膨影響」，提供了央行應對供給側衝擊的操作範本。對沖基金在六週戰爭中遭遇「新冠以來最大損失」（FT報導），大規模去槓桿後的資金重新進場承接。

📐 解讀：聯準會面臨的核心矛盾是：供給側油價衝擊推高CPI，但需求面在高油價下明顯走弱（美國30年房貸申請年減、Delta削減航線）——這是一個典型的滯脹組合。紐西蘭央行的做法（忽略供給側CPI，專注核心需求）是目前最可能的聯準會路徑，但這需要油價確實回落並穩定在90美元以下。
🔗 油價回落至95美元 → 通膨預期降溫（核心PCE壓力減少）→ 聯準會降息空間打開（CME 43%）→ 美債殖利率下行 → 成長股/高估值資產受益 → 美元走弱 → 新興市場資金回流 → 亞洲股匯市同步升值
🔄 週期性 — 週期性：本輪通膨主要由供給側能源衝擊驅動，若停火持續，通膨可在2-3個月內顯著回落，不像2021-22年的需求拉動型通膨需要長期緊縮
🔀 情境A（停火持續，油價穩定在90美元以下）：聯準會Q3降息概率升至70%+，降息25bps，長債殖利率進一步下行20-30bps；情境B（停火破裂，油價重回115美元）：聯準會年內降息概率歸零，市場重新定價「高利率更久」，高收債和REITs承壓
