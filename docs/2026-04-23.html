荷莫茲海峽幾近停擺 — 川普下令擊斃佈雷船，布蘭特突破102美元

發生什麼事

2026年4月23日，伊朗革命衛隊扣押MSC Francesca與Epaminondas兩艘貨輪，荷莫茲海峽同日幾近停擺——僅一艘散裝船LB Energy仍在水道內，成品油輪Ocean Jewel在伊朗開火後停在入口待命。美國總統川普隨即宣布已下令海軍「擊斃任何」試圖在荷莫茲海峽佈雷的船隻，並維持對伊朗港口的海上封鎖。自2月28日美以對伊朗發動戰爭以來，布蘭特原油已突破每桶102美元，美國汽油均價超過4美元（較戰前上漲逾30%）。伊朗官員首度宣布已收到荷莫茲過路費的首批收入，並表示在美國解除港口封鎖前不會恢復和談。加拿大前總理杜魯道在達佛斯表示目前「仍無通往和平的明確路徑」。

📐 解讀：此事件反映的是結構性能源安全危機，而非週期性供給波動。荷莫茲海峽每日原有逾100艘船通行，現已接近完全中斷，且短期內無繞道替代方案。市場目前最大的誤判是把這當成「可控的地緣衝突」，但Citi已警告全球原油庫存在6月底前將觸8年低點。IEA主席Birol的「史上最大能源安全威脅」措辭並非誇大。
🔗 荷莫茲封鎖 → 全球每日原油通過量減少約2,000萬桶（佔全球供給近20%） → 布蘭特油價100-110美元 → 全球運輸、航空、製造業成本全面上升 → CPI黏滯性升高 → 主要央行（Fed、ECB）降息延後 → 高收益債、成長股承壓；石油服務股、替代能源股受益
🏗️ 結構性 — 結構性：荷莫茲封鎖非季節性，替代路線（Sumed管線、Habshan-Fujairah管道）容量合計不足正常量的30%，短期無法替代
⚡ 矛盾訊號：市場矛盾：S&P 500與Nasdaq已於上周創新高，但IEA稱庫存6月觸8年低；股市「shrug off」與原油基本面的分裂，可能是因市場定價停火談判成功的機率仍高，但這預期差風險極大
🔀 如果停火協議在72小時內達成，布蘭特油價可能回落至90-95美元；如果伊朗正式宣布全面控制荷莫茲並向盟國索取「過路費」，油價將測試120美元，美國CPI可能在Q3升至5.0%以上

📐 解讀：這是一個滯脹訊號。油價帶動的CPI上行壓力正在侵蝕消費動能，而Fed降息空間同步收窄。市場近期反彈是因停火預期，但底層的消費信心與企業獲利惡化才是真實基本面。航空業的財測下修只是傳導鏈的起點，不是終點。

🏗️ 結構性 — 週期性疊加結構性：油價飆升是週期性供給衝擊，但消費信心的結構性損傷（若持續6個月以上）會觸發信用週期轉折

伊朗戰爭衝擊美國住宅建材供應鏈 — 泡棉、屋頂、窗戶全面漲價

發生什麼事

美國建材供應商在4月集中公告漲價，涵蓋泡棉絕緣材料、屋頂板材、窗戶、門等主要組件，CNBC調查多家供應商均確認因伊朗戰爭造成原料與運輸成本大幅上升。與此同時，美國30年期固定房貸利率維持在6.0%附近，難突破下行關卡，買氣持續受壓。英國3月通膨率升至3.3%（戰前預期2.7%），住房相關費用（暖氣油、電費）顯著上升。英國財政部赤字雖創三年新低，但分析師警告戰爭將推高政府融資成本。美國主要建商已開始延遲開工或重新評估土地開發計畫。

📐 解讀：建材漲價是戰爭傳導到實體經濟的「第二波」效應，第一波是燃料成本，第二波正在侵蝕建築業利潤率。美國市場供需本已緊張，現在成本端再受打擊，可能讓住宅供給短缺問題更長期化。這對不動產投資人是一個「雙輸」情境：利率不降、成本不降、供給不增。
🔗 伊朗戰爭 → 石化原料（泡棉、PVC）與能源成本上升 → 建材供應商漲價 → 新屋建造成本提升 → 建商開工減少 → 住宅供給惡化 → 租金上升 → 核心CPI加速黏滯 → Fed維持高利率更長
🏗️ 結構性 — 週期性疊加結構性：戰爭是週期性衝擊，但美國住宅供給短缺是10年以上的結構性問題，此次衝擊將延長解決期程
🔀 如果伊朗戰爭在Q2結束，建材成本可能在Q3回落5-10%，建商開工回升，但房貸利率仍高，需求端難以快速復甦；如果戰爭持續至Q3，建造成本繼續上升10-15%，2026年新屋開工年率可能跌破100萬棟（2024年均值約145萬棟）
