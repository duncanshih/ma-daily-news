🔄 新台幣周線連三紅 — 台幣31.4-31.6震盪，外資回補帶動（持續追蹤）

發生什麼事

新台幣4/24收31.516元、升值5.9分、升幅0.19%，本週升值8分（0.25%），周線連3紅。國銀分析央行看好區間31.4-31.6，主因外資匯入回補台股。同時人民幣探半個月低點6.8342（中東地緣不確定性），日圓159附近橫盤（壽險業減碼日債資金外流）。

關鍵數據

新台幣31.516（升5.9分）、人民幣6.8342（半月低）、日圓159橫盤、本週升幅0.25%

📐 解讀：台幣升值是外資追逐台股權值股的副產品，不是基本面驅動。央行對31.4-31.5以下的「軟天花板」很可能在出口商怨聲開始時介入。對台灣壽險業是雙刃劍：台股增值有利、但匯損加大避險成本。
🔗 外資匯入回補台積電 → 新台幣升值壓力 → 央行可能進場 → 但中東能源價格上行 → 美元有支撐 → 央行不必動用太多籌碼即能守住31.5
🔀 情境A：人民幣升至6.6 → 台幣有機會見30.5-30.8；情境B：央行強力守31.5 → 台股外資動能受抑、轉向中小型股

南亞科LPDDR毛利率叩關70%、Vera Rubin即將量產、SK Hynix Q1獲利年增4倍對標案例
