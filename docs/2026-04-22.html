美伊停火僅延3-5天 — 伊朗扣押2艘船，布蘭特油價重破$100

發生什麼事

美國總統川普4月22日宣布對伊朗延長停火，道瓊工業指數早盤大漲411點（+0.8%），費城半導體指數漲1.6%，台積電ADR漲1%。然而停火期限僅延長3至5天（而非市場預期的無限期），並未宣佈廢除海上封鎖。同日伊朗伊斯蘭革命衛隊在霍爾木茲海峽附近扣押2艘貨輪，聲稱報復美國先前扣押伊朗商船。FT報導布蘭特原油隨即重破每桶100美元，Vitol CEO警告伊朗戰爭已使全球市場損失10億桶原油供應。CNN引述2位知情人士稱，川普計畫設定有限時間表，要求伊朗在窗口內提出整合德黑蘭高層意見的正式提案。

📐 解讀：市場反應明顯是情緒面驅動——停火消息引發風險資產上漲，但同日伊朗又採取對抗動作，顯示雙方談判陷入「走一步退半步」的博弈僵局。本質上這是「有限度的衝突管控」而非「通往和平的路線圖」。若3-5天內談判無進展，霍爾木茲風險溢價將快速重回市場，油價可能再次衝高。

🔗 和平預期升溫 → 美元避險溢價消退 → 美元走弱 → 新興市場資金回流、人民幣/台幣/韓元升值 → 台灣出口商匯兌損失增加 → 台股外資持股成本下降（有利外資繼續買進）

發生什麼事

彭博報導引述分析師數據：預計三星電子2026年全年淨利潤將年增400%，SK海力士淨利成長約300%，兩者遠超過Nvidia等AI晶片頭部企業的盈利增速。然而記憶體股的本益比仍顯著低於其他AI相關半導體。市場正激烈辯論：記憶體產業是否已進入「超級週期」、徹底擺脫過去繁榮蕭條交替的週期性軌跡。台灣方面，宇瞻3月單月EPS達6.37元，年增1,713%；景碩3月獲利年增85%；高盛將萬潤目標價上調至1,800元、弘塑至4,500元。

📐 解讀：這是本年度最重要的週期判斷辯論之一。超級週期派認為AI推動HBM（高頻寬記憶體）長期剛性需求，加上中國出口管制限制SK海力士/三星的產能擴張，供需格局根本性改變。反週期派則指出：記憶體是商品，歷史上每次「超級週期」主張出現時，往往就是週期高點。目前台廠宇瞻的1,713%成長率是低基期反彈加上庫存補貨，需區分結構需求與補庫週期。
🔗 HBM需求持續 → 三星/SK海力士資本支出保守（不願再犯2021-2022年過度擴產的錯誤）→ 供給端自律 → 記憶體價格維持高檔 → 台灣記憶體模組廠（宇瞻、威剛）、AI載板廠（景碩）、先進封裝設備廠（萬潤、弘塑）持續受惠 → 外資持續升評台灣半導體供應鏈

生命科學實驗室不動產逆勢回彈 — NIH削資衝擊後，投資者重返市場

發生什麼事

CNBC報導，美國生命科學實驗室不動產（Life Science Lab Real Estate）在2025年遭遇重大打擊——NIH（美國國家衛生研究院）被迫取消數十億美元的研究補助，直接重創各大學與研究機構的實驗室需求。然而2026年Q1起，多個城市的實驗室空置率開始停止惡化，部分機構投資者重新評估進場時機。報導指出這是目前美國商業不動產中少數呈現積極訊號的細分市場，而辦公室、零售等傳統CRE（商業不動產）仍深陷困境。

📐 解讀：生命科學不動產的週期與傳統辦公室完全脫鉤——其需求驅動因素是聯邦研究經費和生技融資活動，而非企業租賃需求。NIH削資是政策性衝擊，具有一次性特質；只要政治環境改變（或新的生技融資潮）就可能出現V型反轉。這是私募不動產基金和基建投資機構值得關注的細分市場。
🔗 NIH補助削減 → 大學實驗室需求萎縮 → 生命科學不動產空置率上升 → 開發商暫停新案 → 供給增量歸零 → 既有存量物件空置率觸頂 → 生技融資回暖後優質物件率先收租
🏗️ 結構性 — 週期性為主：NIH政策是外生性週期衝擊；但生命科學不動產的長期需求（老齡化、AI製藥、基因治療）是結構性的，回調提供長期買點
🔀 如果美國下一屆政府恢復NIH補助、或生技IPO市場在2026年下半年回暖（目前生技IPO已有初步復甦跡象），實驗室租金可能在12個月內回升8-15%；如果政府持續削減科研預算，空置率可能維持高位至2027年以後，不動產估值進一步承壓
