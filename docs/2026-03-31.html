McCormick以450億美元合併聯合利華食品業務 — 消費品整合時代，66億美元新集團重塑全球食品市場

發生什麼事

McCormick（麥考密克）宣布以市值約450億美元（含債務）收購Unilever食品部門，合計形成660億美元的新食品集團，是近年消費品最大M&A交易之一。Unilever此前已宣布全球暫停招聘，並將食品部門分拆視為策略重組核心。交易結構為McCormick以現金+換股方式進行，Unilever保留調味料之外的個人護理、家居清潔業務。同日，Biogen宣布以56億美元收購Apellis Pharmaceuticals（罕見疾病），Lilly以最高78億美元收購Centessa（睡眠障礙），顯示「聚焦核心、剝離非核心」的M&A邏輯在醫藥與消費品同步展開。

📐 解讀：這波M&A浪潮的底層邏輯是「高利率後遺症下的資本效率再優化」——Unilever承受高油價的供應鏈成本、加上中東戰事衝擊消費者信心，選擇在此時出售非核心資產符合「現在賣比等市況更差再賣好」的戰略邏輯。McCormick的整合溢價反映食品調味料業務的定價能力強於多數消費品品類。
🔗 消費品剝離非核心 → 聚焦高毛利品牌 → 收購方槓桿增加（McCormick淨負債比上升） → 信評機構可能下調展望 → 高收益債市場消化新發行量 → 台灣資本市場：對元大/富邦等推動海外M&A顧問業務是潛在案例參考
🏗️ 結構性 — 結構性：全球FMCG整合是長期趨勢，中小型品牌難以獨立承受供應鏈成本上升；週期性因素是高利率環境促使企業加速資產輕量化
⚡ 矛盾訊號：Unilever一方面「全球暫停招聘」（顯示財務壓力），另一方面卻以16億美元賣出（而非整個食品部門的450億美元收益歸Unilever），意味著食品業務估值相對健康，Unilever割捨的理由更多是「聚焦品牌矩陣」而非「賣弱保強」

發生什麼事

金管會統計顯示，壽險業2月底避險比率降至45.1%，創史上新低，較前月減少2個百分點，連兩個月跌破五成。與此同時，21家壽險公司2025年全年稅後純益合計1,805.94億元，年減39.64%（2024年為約2,990億元），並出現4家由盈轉虧（新光人壽合併前、台銀人壽、宏泰人壽、元大人壽）。反向指標：台新新光金法說報喜，2025年稅後純益374億元（年增86%，合併效應），創歷史新高，總經理宣示股利將優於2024年（現金0.9元/股），市場估計今年可能突破1元。另一方面，壽險業1月外幣保單新契約保費達695.36億元，創2019年統計以來同期新高，佔整體壽險新契約44.41%。

關鍵數據

壽險業避險比率45.1%（2月底，歷史低點）、全年稅後純益1,806億（-39.6% YoY）、台新新光金374億（+86% YoY）

📐 解讀：壽險業避險比率走低有兩個解讀：一是主動降低避險成本（外幣貶值時避險成本高），二是被動增加外幣資產曝險（為提高收益率）。在新台幣3月大貶7.29角的背景下，降低避險比率實際上是在承接匯損風險；外幣保單新高反映壽險業者仍在積極擴大外幣資產規模，策略上是在賭美元走強或匯損可通過保費收入彌補。
🔗 避險比率降低 → 壽險業外幣資產匯損風險增加 → 若新台幣繼續升值，壽險業淨值受損 → 資本適足率下降 → RBC（資本適足率）壓力 → 金管會可能要求增資或降低外幣曝險
🔀 若美伊停火後美元走弱（回流新興市場）、新台幣升值，壽險業未避險外幣資產遭受匯損，估影響全年EPS約0.3-0.5元；若新台幣繼續弱，避險比率低的策略獲益
